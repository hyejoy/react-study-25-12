# React Study — 프론트엔드 프리랜서 준비 기록

React → Next.js → 성능 최적화 → 포트폴리오 제작까지!
실전 위주의 학습 과정을 기록합니다.

## 📅 학습 목표

- React 기초 & 심화 문법 복습
- 최신 기술 흐름 반영 (Hooks 중심)
- 컴포넌트 구조 & 상태 관리 감 회복
- Next.js 기반 실전 프로젝트
- SSR/CSR 전략 이해 및 적용
- TailwindCSS/Styled-components 선택 적용
